# planning-poker

オンラインでプランニングポーカーをするサイトのリポジトリです。  
https://main.denqeqveakjkg.amplifyapp.com/

|プランニングポーカーのやり方||
|-|-|
|1. 1~89のいずれかのボタン押下でポイントを決めてください<br><br>※ポイントを決めた人は名前左のemojiが<br>:thinking: から :sunglasses: になります。<br><br>2.全員が :sunglasses: になったら `OPEN` ボタンで各々が出したポイントが表示され、全員の平均が左側に出てきます。<br><br>3. `CLER` ボタン押下で場をリセットして、再度プランニングポーカーをやり直すことができます。|![poker2](https://user-images.githubusercontent.com/38206334/188425836-1a672054-992d-4653-92bf-cbfaa90d5a30.gif)|

## 部屋の作成・参加の方法

|部屋の作成||
|-|-|
|1. ページ右上のボタンからGoogleアカウントでログイン|<img width="1277" alt="スクリーンショット 2022-09-05 18 26 20" src="https://user-images.githubusercontent.com/38206334/188417088-9a619c39-c54c-42f2-ad6f-47df6b96e631.png">|
|2. ログイン後、 `CREATE ROOM` ボタンを押下|<img width="820" alt="スクリーンショット 2022-09-05 18 51 47" src="https://user-images.githubusercontent.com/38206334/188421669-7bbc8827-626b-4b19-a473-377001328f8f.png">|
|3. 名前を指定して `ENTER` ボタンを押下すると作成した部屋に参加します|<img width="1274" alt="スクリーンショット 2022-09-05 18 34 32" src="https://user-images.githubusercontent.com/38206334/188418934-831a096e-aad9-438e-9e52-89aa4c7d0641.png">|
|4. 部屋のURLまたはページ上部にある `ROOM ID` を共有して他のメンバーを招待してください|![copy_roomid](https://user-images.githubusercontent.com/38206334/188420513-c59e0803-9441-4248-8581-08b5d8e2c5e9.gif)|

|部屋への参加||
|-|-|
|トップページにある `JOIN ROOM`　に `ROOM ID`を入力するか、部屋のURLに直接遷移してください. <br><br> ※ 未ログインの場合でも部屋に参加することはできますが、プランニングポーカーに参加するためにはログインが必要です |![join_room](https://user-images.githubusercontent.com/38206334/188422350-60370143-a093-4973-9fb5-afc41d5d92b1.gif)|
